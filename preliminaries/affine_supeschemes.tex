Аффинные суперсхемы, аффинные групповые суперсхемы, \
супералгебры Хопфа, \
дуальность категорий аффинных групповых суперсхем и супералгебр Хопфа.
Все понятия дословно переносятся с книги \cite{jantzen}, также можно посмотреть \cite{affine_quotients}.


Пусть $ \K $ -- произвольное поле.

\definition {
  $\K$-функтором будем называть функтор из категории $ \SAlg $ в $ Sets $.
}

\definition {
  Пусть $ X $ -- $\K$-функтор. $\K$-функтор $ Y $ называется
  подфунктором функтора $ X $, если $ \forall A, A' \in \SAlg
  \forall \phi \in \HomSAlg(A, A') $ выполнены условия:
  $ Y(A) \subset X(A) $ и $ Y(\phi) = X(\phi)|_{Y(A)} $.
}

\definition {
  Для любого семейства подфункторов $ \{Y_i\}_{i \in I} \subset X $ определим
  пересечение $ \bigcap_{\substack{i \in I}} Y_i $ следующим образом:
  $ ( \bigcap_{\substack{i \in I}} Y_i )(A) = \bigcap_{\substack{i \in I}} Y_i (A) $.
}
